== Description ==

WP-LinkChanger is a small script for affiliate link masking. That means all of your affiliate links in a post will automatically be changed to an internal file.

After installation all links of your new posts will look like http://your.domain.tld/exit.php instead of http://evil.affiliatelink.com/ref/id/123

== Installation ==

1. Upload the complete folder `wp-linkchanger` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Write a post with affiliate links inside ;)

== Frequently Asked Questions ==

= Is this plugin working with all affiliate links? =

No. At the moment the plugin only supports links from zanox, affilinet and tradedoubler.

= What about amazon or other networks? =

Amazon links will be supported in a following version. For further networks please leave a comment on the plugin homepage.
